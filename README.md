# webrtc-with-screen-share
! Google Meet
A web application for video meeting + realtime text chat. Developed with Node.js, Express, Peer.js and Socket.io.
# Features
 : Join and leave room
 : Group video meeting
 : Group text chat
 : Unique username checking
 : User is typing...'
 : Limit number of users per room
